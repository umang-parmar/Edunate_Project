# 🌱 EcoAI: Renewable Energy Prediction using Machine Learning

## 📌 Project Overview

EcoAI is an AI-powered sustainability project developed as part of the **Skill4Green Capstone Program**. The project analyzes global renewable energy data and predicts renewable energy trends using **Machine Learning (Random Forest Regressor)**.

The objective is to demonstrate how Artificial Intelligence can support **green decision-making**, renewable energy planning, and sustainable development through data-driven insights.

---

## 🎯 Objectives

- Analyze global renewable energy data.
- Perform data cleaning and preprocessing.
- Explore trends using data visualization.
- Build a Machine Learning model to predict renewable energy share.
- Support Sustainable Development Goals (SDGs) using AI.

---

## 🤖 Machine Learning Model

**Random Forest Regressor**

The model was trained to predict renewable energy trends after performing:

- Data Cleaning
- Missing Value Handling
- Data Type Conversion
- Feature Engineering
- Train-Test Split

---

## 📊 Exploratory Data Analysis (EDA)

The project includes:

- Correlation Heatmap
- Renewable Energy Trend Analysis
- Country-wise Renewable Energy Comparison
- Distribution Plots
- Feature Importance Analysis
- Actual vs Predicted Visualization

---

## 📈 Model Evaluation

The model performance was evaluated using:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

Random Forest Regression was selected because it captures nonlinear relationships more effectively than Linear Regression and provides feature importance for interpretation.

## 🚀 How to Run the Project

### Clone the repository

```bash
git clone https://github.com/your-username/EcoAI-Renewable-Energy-Prediction.git
```

### Navigate to the project folder

```bash
cd EcoAI-Renewable-Energy-Prediction
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the project

Open:

```
EcoAI.ipynb
```

or run:

```bash
python EcoAI.py
```
